A modern, immersive web experience built with React, Vite, and GSAP. 

## 🔗 Repository
[Mohammad-Asfin/ATTACK-ON-TITAN](https://github.com/Mohammad-Asfin/ATTACK-ON-TITAN.git)

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Technologies
- React
- Vite
- GSAP (GreenSock Animation Platform)
- Custom CSS
